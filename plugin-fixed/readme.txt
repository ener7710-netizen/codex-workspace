Самонавчальний SEO-движок для юридичних сайтів. Автоматизує аудит, генерацію контенту та Schema-розмітки з урахуванням специфіки адвокатської діяльності.

== Description ==

SEOJusAI — це інтелектуальна надбудова для WordPress, розроблена спеціально для ніші юридичних послуг. На відміну від стандартних SEO-плагінів, система використовує LLM (OpenAI) для глибокого розуміння контексту сторінок адвокатів.

**Ключові можливості:**
* **Глибокий AI-Аудит:** Перевірка сторінок на відповідність факторам E-E-A-T (Expertise, Experience, Authoritativeness, Trustworthiness).
* **Knowledge Base Engine (KBE):** Система навчається на ваших правках. Якщо ви вказуєте ШІ на помилку, він запам'ятовує це для всіх майбутніх аналізів.
* **Snapshot Safety:** Автоматичне резервне копіювання контенту перед будь-якими змінами, внесеними ШІ.
* **LegalService Schema:** Автоматична генерація JSON-LD розмітки для кращого відображення послуг у пошуковій видачі.
* **Dashboard моніторингу:** Повний контроль над діями ШІ та журналом впливу (Impact Log).

== Installation ==

1. Завантажте папку `seojusai` у директорію `/wp-content/plugins/`.
2. Активуйте плагін через меню 'Плагіни' у WordPress.
3. Перейдіть у розділ **SEOJusAI > Налаштування** та введіть ваш OpenAI API Key.
4. Налаштуйте модулі у вкладці **Модулі**.
